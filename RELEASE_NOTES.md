# Release v1.0.0

## Overview
This release introduces a robust update of the SchoolPortal API. It now supports managing school profiles, institutions, and locations. In addition, comprehensive health checks, telemetry integration, and middleware enhancements for error handling and request logging have been implemented.

## New Features
- Core API endpoints for handling school profiles, institutions, and locations.
- API and database health checks with detailed telemetry reporting.
- OpenTelemetry integration for distributed tracing.
- Centralized error handling and request logging via middleware.

## Improvements and Fixes
- Expanded unit tests covering institutions, locations, profiles, and validations.
- Improved input validations and error handling.
- Enhanced logging and performance monitoring capabilities.

## Known Issues
- Minor performance issues under heavy load remain.
- Further logging enhancements are planned for upcoming releases.
